PGN Generation
==========================================

The module :mod:`lichess.pgn` provides functions to generate PGNs from API games.

Use :data:`~lichess.pgn.io_from_game` to integrate with the `python-chess <https://github.com/niklasf/python-chess>`_ library.
   
.. automodule:: lichess.pgn
    :members: from_game, io_from_game, from_games, save_games
